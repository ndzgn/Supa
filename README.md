# Supa
Application  en ligne de commande de gestion d'un supermarche

# Fonctionnalites
  * Gestion des produits (ajout, suppression, modification)
  * Gestion du stock
  * Gestion des approvisionements
  * Gestion des ventes
  * Statistques des ventes

# Stack technique
  * Java 17
  * MySQL database
